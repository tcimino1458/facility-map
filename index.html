About Facility Navigator

Notes, images, documents, and uploaded floor plans are stored on this device only (browser localStorage). They are not shared with other users or automatically backed up. Clearing browser data will erase all saved content.

Use 🌐 Publish / Share Map in Admin Mode to generate a fully self-contained HTML file with all your data baked in — share it with anyone and they'll see everything instantly.

Use Export Data to back up settings to JSON, and Import Data to restore.
